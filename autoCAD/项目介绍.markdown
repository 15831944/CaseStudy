# AddMyLayer
1.向集合对象添加新成员  也就是在图层表中添加新图层
2.迭代集合对象  也就是循环图层表中所有的图层
3.在集合对象中查找成员  也就是在图层表中查找图层
4.从集合对象中删除成员  也就是在图层表中删除图层
# PositionApplicationWindow

# SizeDocumentWindow
控制活动文档窗口大小：指定大小，最大化，最小化，获取窗口当前状态等
